# Mijie

Mijie 是一个网络解谜游戏网站，提供了非常多样的游戏形式，解谜爱好者可以使用 Mijie 举办丰富多彩的解谜游戏。

支持流程图、排行榜、公告、提交记录等多种功能。

> 本项目最初是为了在 2023 年北京邮电大学百团大战中举办的解谜游戏“哈士奇再现”而开发的。在那之后，2024 年百团大战的解谜游戏 “哈士奇的复仇”、哈士奇壬寅年解谜游戏和癸卯年解谜游戏也使用了 Mijie。
